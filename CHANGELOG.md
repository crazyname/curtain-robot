# 版本记录

本文件记录仓库协作版本。这里的版本号用于描述仓库整体状态，不一定等同于 Python 程序内部的 `PROJECT_VERSION`。

## 0.3.6 - README / 协作说明 / .gitignore 整理

- 将 README 整理为面向 GitHub 使用者和开发者的正式入口文档。
- 清除本机盘符、临时映射盘、本地备份路径、AI 操作规则和历史维护事故记录等不适合公开协作的描述。
- 新增 `CONTRIBUTING.md`，说明分支、PR 范围、提交前检查、版本号和 PR 描述模板。
- 修正根 `.gitignore`，保留多人协作需要的缓存、构建产物、密钥、日志、本地备份和临时文件忽略规则。
