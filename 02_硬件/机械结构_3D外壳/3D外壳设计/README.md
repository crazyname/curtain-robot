# 3D外壳设计文件说明

本目录已按用途整理，避免 SolidWorks 装配体和零件文件混放。

## 01_SolidWorks源文件

存放 `.SLDASM` 装配体和 `.SLDPRT` 零件源文件。

建议优先打开：

- `装配体2.SLDASM`
- `装配体1.SLDASM`
- `打印1.SLDASM`
- `淘宝打印1.SLDASM`

如果 SolidWorks 提示缺少零件，请在本文件夹内查找对应 `.SLDPRT` 文件。

## 02_STL打印文件

存放可直接用于切片或淘宝打印的 `.STL` 文件。

其中 `淘宝打印` 子文件夹保留原有打印文件组合。

## 03_参考图片

存放截图、外观参考图等辅助材料。

## 04_原始压缩包

存放未展开或原始备份压缩包。
