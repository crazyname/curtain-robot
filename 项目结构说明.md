# 智能窗帘项目结构说明

整理时间：2026-05-13

## 顶层分类

```text
智能窗帘/
├─ 01_软件/
│  ├─ Python控制程序/          # 树莓派/Python 控制程序
│  ├─ STM32嵌入式固件/        # STM32CubeMX/Keil 固件工程
│  └─ 微信小程序/              # 微信端控制小程序
├─ 02_硬件/
│  ├─ PCB_AD工程/              # Altium Designer 原理图、PCB、打板文件
│  ├─ 机械结构_3D外壳/         # SolidWorks、STL、结构件模型
│  └─ 立创EDA工程/             # 立创 EDA 工程文件
├─ 03_课程与文档/
│  ├─ 课程提交版本/            # 作业提交版 Python 工程
│  ├─ 汇报材料/                # PPT 汇报文件
│  ├─ 资料表格/                # 表格资料
│  ├─ 项目说明/                # 原始项目 README
│  └─ 目录快照/                # 原始目录树记录
├─ 04_工具/
│  ├─ 立创EDA助手/             # 已解压的立创助手程序
│  └─ 安装包/                  # 立创助手安装包
├─ 05_原始压缩包/
│  ├─ 硬件与固件备份/          # 原始 3D/PCB/固件/小程序压缩包
│  └─ 课程作业备份/            # 原始作业压缩包
├─ .git/
├─ .git_bad_upload_backup/
├─ .gitignore
└─ 项目结构说明.md
```

## 主要文件归属

- Python 主控制程序：`01_软件/Python控制程序/`
- STM32 嵌入式工程：`01_软件/STM32嵌入式固件/My_curtain_robotV2/`
- 微信小程序：`01_软件/微信小程序/My_Curtain_robot_Wechart/`
- PCB 与原理图：`02_硬件/PCB_AD工程/AD_Project/`
- 机械外壳与 3D 打印模型：`02_硬件/机械结构_3D外壳/3D外壳设计/`
- 立创 EDA 工程：`02_硬件/立创EDA工程/ProPrj_1111_2025-11-28.epro`
- 课程作业版本：`03_课程与文档/课程提交版本/作业/`
- 汇报 PPT：`03_课程与文档/汇报材料/汇报.pptx`
- 原始压缩包：`05_原始压缩包/`

## 说明

- 根目录只保留项目级文件和 Git 元数据，具体工程均按类别进入子目录。
- `作业` 与根目录 Python 程序不是完全相同版本，已分别保留。
- `.gitignore` 已同步更新，避免工具安装目录、安装包、压缩包和常见编译产物被误提交。
